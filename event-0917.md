### 在线事件
#### 用户事件
>用户建立连接，用户心跳连接，用户中断连接
```
用户建立连接，心跳，中断->更新维护JVM缓存：（用户id->用户连接集合）
```
#### 群事件
>自己入群,邀请入群,正常退群,被提出群
```
入群，退群->更新维护JVM缓存：（群id->用户id集合)
```

#### 消息发送事件
>消息发送，编辑消息，撤回消息
```
消息发送->推送消息给在线用户，异步更新维护JVM缓存：（群id->用户id集合->（用户收到的最新消息，用户已读的最新消息)）
```
